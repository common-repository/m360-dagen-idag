== Description ==

M360 dagen i dag is a plugin to show info about today evnents in history, in Norwegian language , its connect to nrk and get the info from it.

* add it as widget
* add it as shortcode [m360-dagen]


== Installation ==

Upload the M360 dagen idag plugin to your blog, Activate it, then you can use it as a widget in a sidebar or just a shortcode in where ever you want , use [m360-dagen]
